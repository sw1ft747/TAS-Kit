Chat Commands
Command Description
 !rst  Restart the game round
 !restart  (Re)start speedrun
 !changeon  Enable change level
 !nextlevel  Change level to the next map
 !transit  Get coordinates on the next level
 !hud  Toggle timer
 !fdmg  Toggle fall damage notification
 !pick  Get an action function for the entity you're looking
 !crds  Get your position, angles and velocity
 !zdump  Dump all infected to 'zdump.nut' file in 'left4dead2/ems/tas_kit/' folder
 !trigger  Get function to spawn a trigger at your position
   
 !timer arg1  Set timer countdown; arg1 - value (e.g. 2.5)
 !find arg1  Find an item; arg1 - classname of entity (e.g. pistol_magnum, gascan, pain_pills)
 !xclip arg1  Simulate bot teleportation; arg1 - type: fast/exact/double
   
 !dbg  Toggle developer mode
 !dbg nav  Show navigation mesh
 !dbg items  Glow all items
 !dbg tp  Get action function to teleport yourself
 !dbg trigger  Show triggers
 !dbg start  Start speedrun timer
 !dbg stop  Stop speedrun timer
 !dbg set arg1  Set value of timer; arg1 - value (e.g. 123.5)
 !dbg clear  Remove all spawned messages during debugging
 !dbg tr  Toggle tool for easier creating of triggers
 !dbg dump  Dump created triggers to 'triggers_dump.nut' file in 'left4dead2/ems/tas_kit/' folder
   
 !fillbot  Toggle fill bot
   
 !autosb  Toggle auto spitterboost
 !factor  Change multiplier of jump time
   
 !bhop  Toggle auto bunnyhop
 !bhop2  Toggle host auto bunnyhop for local player
 !aimbot  Toggle aimbot (not local player must bind MOUSE1 to +alt2)
 !aimbot2  Toggle aimbot â„–2, aims after shooting
 !ragebot  Toggle ragebot
 !autostrafer  Toggle simple autostrafer
